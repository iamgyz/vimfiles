\Vim/\Vim/\Vim/\Vim/\Vim/

Bundles
-------

* Pathogen
* NERDTree
* YankRing
* Conque Shell
* snipMate
* Fugitive
* Tabular
* XtermColorTable
* GitGutter

Installation
------------

1. `git clone https://github.com/starbops/vimfiles.git`
2. `cd vimfiles`
3. `ln -s .vimrc ~/.vimrc`
4. `ln -s .vim ~/.vim`
